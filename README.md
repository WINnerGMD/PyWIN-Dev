
PyWIN Dev is a Full Stack example of creating GDPS Hosting based on FastAPI and Vue js.

In the project architecture, you can work with various GDPS Cores on FastAPI; by default, you can install PyWIN Core or RealistikGDPS.

or you can write your own.

> ## Mission
> The project is aimed at helping in the development of hosting and the opportunity to come up with your own idea and implement it.
